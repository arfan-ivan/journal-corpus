# journal-corpus

###Sumber Data:

Google Scholar (menggunakan library scholarly)
DOAJ (Directory of Open Access Journals) dengan detail lengkap
arXiv untuk preprint akademik
PubMed untuk jurnal medis/kesehatan
SINTA (Science and Technology Index Indonesia)
Portal Garuda
Semantic Scholar untuk referensi yang lebih kaya
Repository Universitas Indonesia (UI, ITB, UGM, Unair, ITS)

###Informasi Jurnal:

Judul, abstrak, penulis lengkap dengan afiliasi
Keywords, DOI, URL, PDF URL
Jumlah sitasi, halaman, volume, issue
Penerbit, tahun, bahasa, bidang subjek
Teks lengkap dan daftar referensi
Tanggal pengumpulan
